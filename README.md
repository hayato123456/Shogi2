# Shogi<br>
　DxLibで動かせる将棋を作成している途中です。<br>
<br>
①タイトル<br>
　将棋

②概要<br>
　9x9の盤上に40枚の駒があり、対局者が交互に駒を動かしていき、<br>
  相手の王を動かせなくしたほうの勝ち<br>

③使用言語<br>
　C言語、C++<br>

④開発環境<br>
　VisualStudio2022,DxLib<br>
 
⑤製作期間<br>
　3日間(およそ16時間)<br>

⑥処理<br>
★現在実装されている処理<br>
・駒と将棋盤の描画処理<br>
・マウスでクリックしたところの座標の取得<br>
・駒の入れ替え処理<br>
・プレイヤー側が駒が動かせるかどうかを判定(歩、王、銀、金、桂、と、成銀、成桂、成香)まで<br>
<br>
